# AI FirstOps

A premium, multifunctional React website for AI FirstOps. The site combines a high-end marketing experience with interactive business features: theme switching, site search, solution tabs, ROI planning, pricing toggles, FAQ accordions, and a contact brief form.

## Tech stack

- React
- TypeScript
- Vite
- ESLint
- Custom responsive CSS

## Getting started

```bash
npm install
npm run dev
```

## Production checks

```bash
npm run lint
npm run build
```
