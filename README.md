# Aifirstops

A high-end multifunctional website for Aifirstops, built as a responsive
static experience with no required build step.

## What is included

- Premium hero, navigation, trust metrics, and conversion CTAs
- Filterable solution and showcase sections
- Platform overview, executive insight cards, process timeline, testimonials, and FAQ
- Interactive monthly/annual pricing toggle
- Contact inquiry and newsletter flows with toast feedback
- Light/dark theme toggle with local preference persistence
- Responsive styling for mobile, tablet, and desktop

## Run locally

```bash
npm start
```

Then open <http://localhost:4173>.

## Validate

```bash
npm run lint
```
