# Aifirstops

A premium multifunctional website for an AI-first operations platform. The site is built as a lightweight static
frontend with polished responsive design and interactive business features that can later connect to a backend, CMS,
checkout, CRM, or analytics stack.

## Features

- High-end responsive landing page with sticky navigation, scroll progress, and theme switching
- Product, solutions, marketplace, process, pricing, resources, FAQ, and contact sections
- Interactive solution tabs, marketplace filtering, annual/monthly pricing toggle, ROI calculator, and testimonial carousel
- Modal preview, simulated contact/newsletter submissions, toast notifications, and back-to-top control
- Accessible semantic HTML structure with skip link, form labels, keyboard-friendly controls, and reduced-motion support

## Run locally

```bash
npm start
```

Then open <http://localhost:4173>.
