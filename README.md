# AI First Ops Website

A premium multifunctional website for AI First Ops, built as a fast static Vite project.

## What is included

- High-impact landing page with hero, proof metrics, and dashboard preview
- Solutions, platform, portfolio, pricing, resources, testimonials, FAQ, contact, and newsletter sections
- Responsive design for mobile, tablet, and desktop
- Dark/light theme toggle with local preference persistence
- Interactive portfolio filters, resource search, pricing toggle, FAQ accordions, and form validation
- Self-contained SVG favicon and CSS-driven visual system

## Getting started

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Project structure

```text
.
|-- index.html
|-- favicon.svg
|-- package.json
`-- src
    |-- main.js
    `-- styles.css
```

## Customization notes

- Update copy and sections in `index.html`.
- Adjust colors, spacing, and component styling in `src/styles.css`.
- Extend behavior such as forms, filtering, and analytics in `src/main.js`.
- Connect forms to a backend, CRM, or serverless function before using them for production lead capture.
