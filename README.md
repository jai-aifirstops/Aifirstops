# Aifirstops

A premium, multifunctional static website for an AI-first business platform. The site is built with plain HTML, CSS, and JavaScript so it can be opened directly in a browser or served by any static host.

## What is included

- Luxury responsive landing page design
- Hero section with animated counters and dashboard preview
- Solutions, platform, workflow, pricing, testimonials, FAQ, and contact sections
- Theme toggle with local preference storage
- Mobile navigation
- Interactive workspace tabs
- Monthly/annual pricing switch
- Testimonial carousel
- Contact and newsletter form status handling

## Local preview

Open `index.html` in a browser, or serve the folder with any static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
