# Aifirstops

A high-end multifunctional static website for an AI-first operations brand. The site is designed as a polished public front door with conversion messaging, service education, pricing, an ROI calculator, FAQ, contact intake, responsive navigation, dark/light mode, and front-end interaction details.

## What is included

- Premium responsive landing page
- AI operations hero and dashboard-style visual preview
- Feature grid and solution tabs
- Interactive ROI estimator
- Testimonials and pricing cards
- FAQ accordions
- Contact form with client-side success feedback
- Dark/light theme toggle
- Scroll reveal animations and back-to-top control

## Run locally

```bash
npm run serve
```

Then open <http://localhost:4173>.

## Validate

```bash
npm run check
```
