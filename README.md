# Aifirstops Portfolio

A responsive single-page portfolio for AI-first operations, automation, and product delivery work.

## Preview locally

Open `index.html` in a browser, or serve the folder with any static file server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Customize

- Update project details and contact links in `index.html`.
- Adjust colors, spacing, and responsive behavior in `styles.css`.
- Keep navigation behavior and the dynamic footer year in `script.js`.
